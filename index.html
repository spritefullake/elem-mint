Elem-mint

Create DOM elements on the fly with strings

View on GitHub Download .zip Download .tar.gz

Create

Use Elem-mint to create DOM objects on the fly -- strings most certainly attached.

$E("div#first[name=thefirst]");// <div id="first" name="thefirst"></div>
yes, it's that easy, so why not try it?
  • Ultra lightweight
  • No bloat
  • Quick
  • Simple
  • Reusable
